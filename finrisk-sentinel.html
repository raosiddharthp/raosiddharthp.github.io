Agentic Design Patterns & Technical Moats

1. Collaborative Reasoning (Hierarchical Handoff)

Unlike a linear pipeline, Sentinel uses Dynamic Task Delegation. If the Anomaly Detective finds a high-risk score, it doesn't just pass a flag; it triggers a "Sub-Crew" task. The KYC Researcher is summoned to perform a deep-dive into the beneficiary‚Äôs history using Vertex AI Extensions to query external AML watchlists in real-time.

The "Human-in-the-Loop" (HITL) Guardrail

In banking, 100% autonomy is a regulatory risk. We implement a Self-Correction Loop where .

  • Agent Critique: Before a SAR is finalized, a "Compliance Auditor Agent" (using a stricter, zero-temperature Gemini model) reviews the narrative against a "Policy-as-Code" vector database.
  • Escalation Trigger: If the confidence score of the swarm falls below 85%, the system automatically generates a "Case Brief" for a human investigator, rather than making an autonomous decision.

3. Deterministic Backbone with Agentic Flexibility

To satisfy the EA (Enterprise Architecture) requirement for auditability, the swarm operates within a LangGraph-based State Machine.

  • Stateful Memory: Every "thought" and "tool call" is logged as a JSON artifact in BigQuery, providing a 100% transparent audit trail of why an agent made a specific recommendation.
  • Event-Driven Execution: Agents are hosted on Cloud Run, scaling to zero when idle and bursting to handle spikes during high-volume periods (e.g., global market volatility).

Key Performance Indicators (KPI) Realized

  • Reduced Cognitive Load: By delegating data gathering to agents, human analysts spend 0% of their time on data retrieval and 100% on high-value risk judgment.
  • SAR Quality Consistency: Using Gemini's long-context window (2M tokens), the Narrative Architect can ingest months of transaction history to find patterns that a human investigator might miss, leading to the 85% faster generation mentioned in our strategy.
